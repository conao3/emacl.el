* Overview

emacl.el provides a simple and lightweight email client for Emacs.

* Requirements

- Emacs 26.1 or later
- [[https://github.com/chuntaro/emacs-async-await][async-await]] 1.1 or later

* Installation

** Using leaf.el

#+begin_src emacs-lisp
(leaf emacl :ensure t)
#+end_src

** Using package.el

#+begin_src emacs-lisp
(package-install 'emacl)
#+end_src

** Manual Installation

Clone this repository and add it to your load path:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/emacl.el")
(require 'emacl)
#+end_src

* Usage

#+begin_src emacs-lisp
(require 'emacl)
#+end_src
